CT QC Copilot: Clinical Implementation of an Automated Quality-Control Decision-Support System

Chengshuai Yang
NextGen PlatformAI C Corp
physics.med-ph cs.AI
Submitted
February 2026
Paper ID
aiXiv:2502.005

Abstract

We describe the clinical implementation of an automated decision-support system for CT quality control. The system integrates automated phantom image analysis, statistical process control (SPC) trending, and structured decision recommendations into a unified workflow. Per-scanner analysis time is reduced from approximately 67 minutes (manual review) to 4.2 minutes (automated analysis + physicist review) -- a 94% decrease in physicist effort. Western Electric SPC rules applied to longitudinal QC data identify scanner degradation 3-6 months before threshold exceedance in 4 of 30 simulated scanners, enabling predictive maintenance. Metric agreement is within 1.2 HU for CT number accuracy and 0.10 mm for geometric measurements compared to manufacturer reference values. The system provides structured pass/warning/fail recommendations with supporting evidence, maintaining physicist authority while eliminating routine manual measurement tasks.

Keywords: CT quality control, Decision support, Medical physics, Statistical process control, Automation, Clinical implementation
